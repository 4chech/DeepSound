WRITE UP
Имеем 4 одинаковые композиции, первым делом - необходимо посчитать хэш одним и тем же алгоритмом(был использовал алгоритм хэширования MD5) от 4 файлов - у двух аудиофайлов хэши будут отличаться - соответственно, в эти файлы были внесены какие-то изменения.
 
Также стоит обратить внимание на название таска - "Deepound".

В гугле можно найти утилиту, deepsound, позволяющую прятать информацию в аудиофайлах.

Ссылка на гитхаб https://github.com/Jpinsoft/DeepSound

В файлах 3.wav и 2.wav лежит необходимая для решения таска инфоромация. flag.txt и real_flag.txt.

 

После нажатия на extract secret files flag.txt будет содержать в себе пароль, для расшифровки спрятанного файла из 2.wav.
 
Следует лишь ввести этот пароль для расшифровки информации из 2.wav.
 

После ввода пароля и экспорта секретного файла из 3.wav можно будет получить текстовый файл с флагом
